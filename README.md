# Song Xudong Personal Website

Personal academic blog and portfolio powered by Hexo and the AnZhiYu theme.

## Local Development

```powershell
npm install
npm run dev
```

Open <http://localhost:1313/>.

## Build

```powershell
npm run build
```

## Content

- Posts: `source/_posts/`
- About page: `source/about/index.md`
- Theme config: `_config.anzhiyu.yml`
- Site config: `_config.yml`

## Deployment

GitHub Actions builds the site and deploys it to GitHub Pages.
